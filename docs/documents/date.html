formatTime(date, format = 'YYYY-MM-DD', invalidText = '--')
日期格式化

参考: dayjs.format

Arguments

date ( Number | String | Date ) : 日期
[format='YYYY-MM-DD'] ( String ) : 格式化
[invalidText='--'] ( String ) : 解析错误时的返回

Returns

( String ) : 日期格式文本

Example

暂无
formatTime(1628659676589);
// => '2021-08-11'
formatTime(1628659676589, 'YYYY-MM-DD HH:mm');
// => '2021-08-11 13:27'
formatTime(1628659676589, 'YYYY-MM-DD HH:mm:ss');
// => '2021-08-11 13:27:56'
formatTime(new Date('2021-08-11 13:27:56'));
// => '2021-08-11'
formatTime('2021-08-11 13:27:56');
// => '2021-08-11'
formatTime(null);
// => '--'
