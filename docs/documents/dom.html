classNames(...args)
classNames

See

https://www.npmjs.com/package/classnames

Arguments

args (...*) : 每个className的描述

Returns

( String ) : className 字符串

Example

暂无
classNames('foo', 'bar')
// => 'foo bar'
classNames('foo', { bar: true })
// => 'foo bar'
classNames({ 'foo-bar': true })
// => 'foo-bar'
classNames({ 'foo-bar': false })
// => ''
classNames({ foo: true }, { bar: true })
// => 'foo bar'
classNames({ foo: true, bar: true })
// => 'foo bar'

convertCssom(cssom = {})
给cssom加上单位px

Arguments

[cssom] ( Object ) : css 对象

Returns

( Object ) : 带有'px'单位的 cssom

Example

暂无
convertCssom({ width: 100, height: 200 })
// => { width: '100px', height: '200px' }
convertCssom({ width: 100, minHeight: 100, marginTop: 10, paddingBottom: 10 })
// => { width: '100px', 'min-height': '100px', 'margin-top': '10px', 'padding-bottom': '10px' }
suffixClassNames(baseClassName = '', suffixConfig = {}, config = {})
给 className 加后缀

适用于开发组件库时, 给className加作用域

Arguments

[baseClassName=''] ( String ) : 基准 ClassName
[suffixConfig={}] ( Object ) : classNames 对象
[config={separator: '-'}] ( Object ) : classNames 对象

Returns

( String ) : className 字符串

Example

暂无
suffixClassNames('table', { bordered: true, shadow: false })
// => 'table table-bordered'
