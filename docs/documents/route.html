search(key = '')
获取 search 参数

Arguments

key ( String ) : [description]

Returns

( String | Object ) : [description]

Example

暂无
// 假设当前的url为 http://aa.com/abc/d?a=1&b=2
search()
// => { a: '1', b: '2' }

search('a')
// => '1'

stringifyUrl(url = '', params = {})
拼接url

Arguments

[url=''] ( String ) : 基础url
[params={}] ( Object ) : query参数

Returns

( String ) : 拼接的完整url

Example

暂无
stringifyUrl('', { a: 1 })
// => '?a=1'
stringifyUrl('abc', { a: 1 })
// => 'abc?a=1'
stringifyUrl('http://aa.com/abc/d', { a: 1 })
// => 'http://aa.com/abc/d?a=1'
linkTo(url = '', params = {}, options = {})
跳转页面

Arguments

url ( String ) : 基础url
[params={}] ( Object ) : query参数
[options={}] ( Object ) : a链接的属性

Returns

(*) : 跳转页面

Example

暂无
linkTo('http://aa.com/abc/d', { a: 1 })
// => 打开页面 http://aa.com/abc/d?a=1
linkTo('http://aa.com/abc/d', { a: 1 }, { target: '_blank' })
// => 新标签打开页面 http://aa.com/abc/d?a=1
updateUrlQuery(params = {}, url = '')
更新 url 某个参数

Arguments

params ( Object ) : 新的query参数
url ( String ) : 基础url

Returns

( String ) : 更新query后的完整url

Example

暂无
updateUrlQuery({ a: 1 }, '');
// => '?a=1'
updateUrlQuery({ a: 2 }, 'http://aa.com/abc/d?a=1')
// => 'http://aa.com/abc/d?a=2'
getParams(str, key)
获取参数

Arguments

str ( String ) : query字符串
key ( String ) : query的key

Returns

( String | Object ) : 完整的query对象或者单个的query值

Example

暂无
getParams('a=1&b=2&c&d=2&d=3')
// => { a: '1', b: '2', c: null, d: ['2', '3'] }
getParams('a=1&b=2&c&d=2&d=3', 'a')
// => '1'
parseUrl(url = '')
解析 url

Arguments

[url=''] ( String ) : url字符串

Returns

( Object ) : { 'protocol', 'host', 'pathname', 'port', 'search', 'hash', 'origin', 'hostname' }

Example

暂无
parseUrl('http://aa.com/abc/d?a=1');
// => {"protocol": "http:", "host": "aa.com", "pathname": "/abc/d", "port": "", "search": "?a=1", "hash": "", "origin": "http://aa.com", "hostname": "aa.com"}
getFullUrl(url = '')
获取完整 url

Arguments

url ( String ) : 相对路径

Returns

( String ) : 完整的url

Example

暂无
// 假设当前的url为 http://aa.com/abc/d?a=1&b=2
getFullUrl('/abc')
// => http://aa.com/abc
